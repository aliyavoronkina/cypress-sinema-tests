## Issue 1: Логин в админку не работает

- **Тест**: "Успешный логин с валидными данными (happy path)"

- **Шаги для воспроизведения**:
    1. Перейти на http://qamid.tmweb.ru/admin/
    2. Ввести email: qamid@qamid.ru
    3. Ввести пароль: qamid
    4. Нажать кнопку "Авторизоваться"
- **Ожидаемый результат**: Открывается панель администратора с разделами управления
- **Фактический результат**: Страница перезагружается, но форма логина остается видимой
- **Влияние**:
    - Невозможно протестировать функциональность админ-панели
    - Невозможно получить названия залов для теста бронирования
- **Статус**: Тест оставлен ПАДАЮЩИМ согласно требованиям задания
